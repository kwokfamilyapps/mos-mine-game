Step 1：開 Project + 起 Main Level

1.1 雙擊開 Project
C:\Users\verno\Documents\Unreal Projects\MineTrain\MineTrain.uproject
UE5 會問你要咪重建，㩒 Yes 等佢 compile。
1.2 開新 Level
File → New Level → Empty Level → Save As → MainLevel
Save 喺：Content/Maps/MainLevel.umap

Step 2：整 Landscape 地形

2.1 開 Landscape Mode
Toolbar → Landscape Mode（山仔icon）→ New Landscape
設定數值
Section Size7x7 (255x255 quads)
Scale100 x 100 x 50
LocationZ = -50（地平線）
2.2 雕塑地形
Sculpt Tool → 筆刷 Size 20 → 力度 0.5
  • South-East 角升高（240ML山頂）
  • North-West 角放低（耀安邨海邊）
  • 中間整蜿蜒河谷

Step 3：BP_TrackSpline（路軌）

3.1 建立 Blueprint
Content Browser → 右鍵 → Blueprint Class → Actor
改名：BP_TrackSpline
3.2 加 Spline Component
Components → Add → Spline Component → 改名 TrackSpline
Details 入面㩒 + 加 Spline Points，由山頂拉到海邊
3.3 加路軌 Mesh
用 SplineMesh Component 動態生成路軌
Event BeginPlay → For Loop (0 to SplinePointsNum-2) → Get Location at Spline Point (i) → Get Location at Spline Point (i+1) → Create SplineMesh Component → Set Start/End Position & Tangent

Step 4：BP_MineCart（礦車）

4.1 建立 Blueprint
Content Browser → 右鍵 → Blueprint Class → Pawn
改名：BP_MineCart
4.2 Components
ComponentAdd設定
Root預設-
CartMeshStatic Mesh揀 Shape_QuadPyramid / 自訂礦車模型
SpringArmSpringArm Component位置 Z=150, Y=-200, Lag=0.3
CameraCamera Component跟 SpringArm
CollisionBox CollisionSize 100x60x80
4.3 變量 Variables
名稱類型預設
TrackSplineRefActorNone
SpeedFloat0
MaxSpeedFloat300
DistanceAlongFloat0
bIsMovingBooleanfalse
bAutoModeBooleantrue
4.4 Event Tick（核心移動邏輯）
Event Tick → [DeltaTime] Branch (bIsMoving OR bAutoMode) True → Speed = (bAutoMode ? 150 : Speed) Distance = Distance + Speed * DeltaTime Clamp Distance (min:0, max:SplineLength) Get Location at Distance Along Spline [TrackSplineRef] Set Actor Location + Set Actor Rotation False → (do nothing)
4.5 鍵盤 Input
Project Settings → Input → Bindings W → 加速 (Speed +50) S → 減速 (Speed -50) Space → 自動模式開關 (bAutoMode = !bAutoMode)

Step 5：BP_NarratorStation（語音站）

5.1 建立 Blueprint
Blueprint Class → Actor → 改名 BP_NarratorStation
5.2 Components
Box Collision → Trigger Volume → Size 200x200x200 Billboard → 顯示站名
5.3 變量
名稱類型
StationNameText
NarratioTextText
AudioFileSound Wave
WaitTimeFloat (default: 30)
5.4 觸發邏輯
OnActorBeginOverlap → Stop the cart (bIsMoving = false) Play narration audio Display subtitles Wait WaitTime seconds Resume (bIsMoving = true)
5.5 放置6個車站 Level
站位置WaitTime
240ML 礦洞入口Spline 起點20s
選礦廠路程1/430s
恩光堂路程2/530s
🌅 天光墟路程1/245s
平安橋路程2/320s
耀安邨碼頭終點自由

Step 6：天光墟場景（清晨市集）

6.1 晨光設定
置入 DirectionalLight → Rotation: -30度 → Color: 暖色 1.0,0.8,0.6 置入 Sky Atmosphere → 調到日出 mode 建議加 Exponential Height Fog → 低層霧效果
6.2 檔口擺設
檔口Mesh擺位
菜檔Box + 螺旋儀路邊
魚檔Box + 魚 Static Mesh菜檔隔離
豬肉檔Box + 紅色 Block中間位
燒餅佬Box + 圓形 Mesh教堂門口
雲吞麵Box + Steam Particle尾位
士多Box + 彩色點綴路口
6.3 環境聲
Add Ambient Sound → 天光墟叫賣聲（mp3） 可以用之前我生成嘅 AI voice stop6-market.ogg

Step 7：打包測試

7.1 手機測試
Toolbar → Platforms → Android / IOS → Project Settings → Android → 包名設定 → Bundle Identifier: com.minecart.tramtour
7.2 PC 測試
Toolbar → Launch → Windows → 直接 Play
