🎯 遊戲主題

《馬鞍山礦車大冒險》係一款以 1950-70年代馬鞍山鐵礦場 為背景嘅手機冒險遊戲。玩家控制一架礦車，穿越礦洞、礦村、碼頭，體驗當年礦工嘅真實工作環境。

🏆 核心目標：
透過遊戲令玩家認識香港唯一大型鐵礦場嘅歷史，以真實場景還原吸引本地居民同歷史愛好者。

🎮 遊戲類型： 橫向卷軸冒險 + 避障 + 收集
📱 平台： HTML5 手機網頁遊戲（browser開到，唔使安裝）
👥 目標玩家： 馬鞍山居民、香港歷史愛好者、中學生、親子
🎨 美術風格： 像素風（pixel art）或手繪風格，還原70年代懷舊感

📖 主線故事

你係一個 1970 年代嘅年輕礦工，第一天返工就要由山頂嘅 240ML 礦洞 出發，穿過地下坑道網絡，去到 110ML 運輸坑道 坐上電動火車，將鐵礦石運送到 碼頭 出口去日本。

沿途你會經過礦場各個角落，見到礦工工作嘅場景，仲要避開各種障礙。完成礦場任務後，你要由山頂穿越 礦村四區，最後安全抵達 耀安邨。

⛰️ 第一章：240ML 礦洞入口

🏛️ 歷史場景 🎮 向下滑行 難度：⭐⭐

🏗️ 場景描述（忠於歷史）

你企喺 240ML 礦洞口 前面。呢個係一個方形混凝土入口，上面寫住「240M.L.」，入口上方有兩個大圓洞——呢啲係當年嘅抽氣扇位置。洞口嘅鐵門已經打開，裏面黑漆漆一片，只聽到水滴聲同遠處嘅鑿石聲。

空氣中瀰漫著火藥味同泥土味，你深呼吸一口，揸實礦車嘅控制桿，出發！

🎮 玩法

  • 礦車控制： 手機左右傾斜/㩒屏幕左右控制礦車喺路軌上嘅平衡
  • 避開障礙： 塌落嘅岩石、遺留嘅工具、地下積水
  • 速度控制： 當年礦車用木棍剎車，遊戲中靠㩒屏幕控制速度（太快會出軌！）
  • 收集： 鐵礦石碎片（加分用）、礦工遺留嘅物品

👀 沿途見到嘅礦工工作場景

  • 👷 鑽井隊用風鑽喺岩壁上鑽孔（共15人）
  • 💥 爆破隊安裝硝酸甘油炸藥（共15人）
  • 🔨 揀礦工用小鎚輕敲礦石，以顏色（灰黑色）、重量（較重）、質感（易碎）分辨鐵含量
  • 🪨 礦工用V型礦斗卡車沿鐵軌運送礦石，用木棍剎車控制速度
  • 🐦 角落嘅金絲雀籠——礦工用金絲雀檢測有毒氣體

✅ 過關要求

成功從240ML滑行到坑道底部，收集至少10塊鐵礦石，避免撞到障礙物超過3次

🕳️ 第二章：豎井飛躍 — 240ML→110ML

🏛️ 歷史場景 🎮 跳躍 + 節奏 難度：⭐⭐⭐

🏗️ 場景描述

你進入咗礦洞嘅核心地帶——呢度係棋盤式坑道網絡。頭頂有燈泡發出微弱黃光，四周係坑道支柱。你會經過 192ML、144ML 水平層，見到礦石從上層經 豎井（垂直通道） 傾倒落嚟，以風壓開關閘控制，似一個巨型存礦倉。

🎮 玩法

  • 跳躍豎井： 要喺啱嘅時機跳過豎井，唔係就會跌落去
  • 礦石雨： 上層傾倒礦石落嚟時要避開（節奏遊戲元素）
  • 風壓閘： 要喺閘門關閉前衝過去
  • 斜道纜車： 部分路段要跳上斜道嘅纜車，搭一段順風車

✅ 過關要求

成功到達110ML水平層，坐上電動火車

🚂 第三章：110ML 運輸坑道

🏛️ 歷史場景 🎮 橫向衝刺 難度：⭐⭐⭐

🏗️ 場景描述

你終於到達 110ML 運輸坑道。呢度係海拔110米嘅主運輸層，坑道長約 2,200米，係礦場嘅心臟地帶。一架天線式電動牽引機車喺路軌上等你——每次可拖帶 10幾架載礦卡車，運載量達數十噸！

你坐上機車，拉動控制桿，列車沿路軌向洞口方向駛去。沿途經過支撐柱、通風管道、仲有豎井嘅底部出口。

洞口係半圓拱形，上面寫住「110M.L.」。出口設有 自動磅重儀 記錄產量。

🎮 玩法

  • 電動火車駕駛： 控制火車速度，避開路軌上嘅碎石
  • 換軌： 遇到分岔路要揀啱路線（有死路）
  • 通風扇： 經過大型通風扇時要低頭避開（快速㩒屏幕蹲下）
  • 收集礦石： 卡車上嘅鐵礦石會跌出嚟，要撿返

✅ 過關要求

成功駛出110ML洞口，磅重儀記錄到至少20噸礦石

🏭 第四章：選礦廠 → 碼頭區

🏛️ 歷史場景 🎮 搬運 + 限時 難度：⭐⭐

🏗️ 場景描述

礦石從110ML運出後，送到旁邊嘅 選礦廠（建於1954年，耗資十四萬元）。呢個廠房24小時運作，機器聲震耳欲聾！

選礦流程一路展開：

  1. 碎礦機 — 三組機械逐步將礦石打碎至直徑5厘米
  2. 振動篩 — 分離粗粒
  3. 乾選機（電磁） — 挑選高鐵礦粒
  4. 球磨機 + 棒磨機 — 磨成幼砂（不斷加水）
  5. 濕選機（電磁） — 吸起鐵砂，純度由37%提升至57-59%
  6. 脫磁 → 沉澱 → 真空脫水 — 變成精礦粉
  7. 運輸帶 — 送至精礦堆積場 → 再送去碼頭

🎮 玩法

  • 限時運輸： 喺指定時間內將礦石搬運到唔同嘅機器中
  • 小遊戲： 操作磁鐵吸起鐵砂，避開石頭（似夾公仔機）
  • 碼頭裝船： 用運輸帶將精礦送上躉船，再吊上貨輪「大寶山號」或「大寶石號」
  • 避車： 當年兒童返學要避開運礦車，遊戲中要避開來回嘅夾斗貨車

✅ 過關要求

成功將10噸精礦裝上貨輪，完成今日運送任務

🏘️ 第五章：礦村四區探索

🏛️ 歷史場景 🎮 收集 + 解謎 難度：⭐

🏗️ 場景描述

完成礦場工作後，你決定探索呢個住了接近一萬人嘅礦村。礦村分為四個區域，每區住著唔同嘅社群：

📍 山頂區（北方人聚居）

呢度係礦村嘅經濟核心。有 信義會恩光堂（1952年建成，香港信義會第一所自建禮拜堂，正面山牆上有小鐘樓，歌德式尖拱窗，聖所後牆有「耶穌帶領羊群」壁畫）；天主堂聖若瑟堂（1952年，現代風格+歌德式尖拱門窗，混凝土結構，金字屋頂）；仲有 聖若瑟小學校、修道院、神父樓（入口有聖母抱聖嬰浮雕）。

📍 馬鞍橋區（客家人聚居）

位於馬鞍山郊野公園燒烤場附近。呢度有 馬鞍橋 遺跡，周圍係矮小石屋同菜園。

📍 上下半山/潮洲村區（潮州人聚居）

沿山路往下走，左邊路口進入就係潮州村。潮州工隊（如張松秋工隊）就係呢一帶。

📍 碼頭區（廣東人聚居）

即係今日嘅 恒安邨、耀安邨 位置。當年呢度有民居、車房、貨倉、木廠、仲有市集。礦石就係喺呢度用躉船同貨輪運去日本。

🎮 玩法

  • 探索收集： 喺四區走動，收集歷史文物（舊照片、礦工工具、教會文件）
  • 互動解謎： 教會場景有謎題要解（例如排列聖經經文、敲鐘順序）
  • 人物對話： 遇到村民可以對話，了解礦村生活故事
  • 小遊戲： 幫村民送貨、修理礦工屋企
  • 📸 拍照收集： 用手機鏡頭（遊戲內）影低歷史建築，解鎖圖鑑

可收集文物（參考真實歷史）： 信義學校舊課本、礦工頭燈、硝酸甘油炸藥包裝、大公洋行文件、日本工程師測量圖、礦車木剎車棍。

✅ 過關要求

收集至少8件文物，訪問3個村民，解鎖教堂隱藏房間

🌊 第六章 BOSS戰：耀安邨洪水逃亡

完成礦村探索後，你要由山頂安全到達山腳嘅 耀安邨。但呢個時候—— 每年雨季嘅洪水突然爆發！ 💀

🎮 BOSS戰玩法

  • 場景： 由鞍山探索館（山頂）一路向下跑到耀安邨停車場
  • 洪水追趕： 畫面底部洪水不斷上升，你要一路向上跳
  • 障礙物：
    • 🌊 海邊吹上嚟嘅鹹水風 — 強風會吹斜你，要𨅝低
    • 🚗 大佬亂cut線 — 的士突然從側邊衝出
    • 🚲 耀安邨單車仔 — 學生踩單車橫衝直撞
    • 🅿️ 停車場洪水 — 積水區要跳過，唔係會慢落嚟
    • 🪨 山泥傾瀉 — 礦場遺址嘅碎石滾落
    • 🚧 路陷 — 當年礦洞可能導致地面下陷

🎮 特殊機制

  • 體力值： 跑得太快會消耗體力，要收集路邊嘅「腐乳饅頭」補充（當年礦工嘅食物）
  • 分數加成： 路上有你之前收集嘅文物，撿到可以加分
  • 水位警示： 畫面邊緣會變色——由黃→橙→紅，代表洪水迫近

🏆 通關條件

成功逃到耀安邨安全位置，洪水淹浸前到達終點
