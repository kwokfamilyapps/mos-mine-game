🎯 遊戲主題

《馬鞍山礦車大冒險》係一款以 1950-70年代馬鞍山鐵礦場 為背景嘅手機冒險遊戲。玩家控制一架礦車，穿越礦洞、礦村、碼頭，體驗當年礦工嘅真實工作環境。

🏆 核心目標：
透過遊戲令玩家認識香港唯一大型鐵礦場嘅歷史，以真實場景還原吸引本地居民同歷史愛好者。

🎮 遊戲類型： 橫向卷軸冒險 + 避障 + 收集
📱 平台： HTML5 手機網頁遊戲（browser開到，唔使安裝）
👥 目標玩家： 馬鞍山居民、香港歷史愛好者、中學生、親子
🎨 美術風格： 像素風（pixel art）或手繪風格，還原70年代懷舊感

📖 主線故事

你係一個 1970 年代嘅年輕礦工，第一天返工就要由山頂嘅 240ML 礦洞 出發，穿過地下坑道網絡，去到 110ML 運輸坑道 坐上電動火車，將鐵礦石運送到 碼頭 出口去日本。

沿途你會經過礦場各個角落，見到礦工工作嘅場景，仲要避開各種障礙。完成礦場任務後，你要由山頂穿越 礦村四區，最後安全抵達 耀安邨。

🏛️ 歷史場景 🎮 向下滑行 難度：⭐⭐

🏗️ 場景描述（忠於歷史 + 鞍山探索館壁畫藍本）

你企喺 240ML 礦洞口 前面。呢個係一個方形混凝土入口，上面寫住「240M.L.」，入口上方有兩個大圓洞——呢啲係當年嘅抽氣扇位置。

洞口旁邊有一個方形建築，上面寫住「一箱全家 240ML」——係礦場嘅儲水設施。

🪪 警衛室 — 入洞安全程序

🚦 每次入礦前必做！

每個礦工進入240ML之前，必須先去 警衛室，喺工作通告板上掛上自己嘅名牌：

🔴 紅色 = 已入礦洞 🟢 綠色 = 已離開礦洞

呢個制度係用嚟追蹤邊個仲喺礦洞入面——收工時如果有紅色牌未翻轉，就要入洞揾人！

🎮 遊戲玩法

  • 每次入洞前： 一定要去警衛室將名牌翻做🔴紅色
  • 每次出洞後： 記得返警衛室翻做🟢綠色
  • ⚠️ 懲罰機制： 如果收工前唔記得翻轉名牌，警衛會通報，扣你分數/延遲收工
  • 🔴 突發事件： 收工時發現有人嘅名牌仲係紅色但人已經出咗——要四圍揾佢（支線任務）

  • 避開障礙： 塌落嘅岩石、遺留嘅工具、地下積水

🕳️ 第三章：豎井飛躍 — 240ML→110ML

🏛️ 歷史場景 🎮 跳躍 + 節奏 難度：⭐⭐⭐

🏗️ 場景描述（參考鞍山探索館壁畫結構）

你進入咗礦洞嘅核心地帶——棋盤式坑道網絡。頭頂有燈泡發出微弱黃光。

由240ML沿 斜路 向下走到 192ML平台——呢度係運輸中轉站，礦工推著載滿鐵礦石嘅車經過。再沿斜路落到 144ML平台——呢度係採礦前線，礦工用風鑽鑽孔、用鎚揀礦。

由144ML再落就唔係斜路，而係一條 垂直豎井（直井） 直達110ML。礦石就係由呢條直井掉落去，然後喺110ML坐上電動火車運出礦洞。

礦工就係靠呢啲斜路同直井喺各層之間來回，尋找鐵礦石。

🎮 玩法

  • 跳躍豎井： 要喺啱嘅時機跳過豎井，唔係就會跌落去
  • 礦石雨： 上層傾倒礦石落嚟時要避開（節奏遊戲元素）
  • 風壓閘： 要喺閘門關閉前衝過去
  • 斜道纜車： 部分路段要跳上斜道嘅纜車，搭一段順風車

✅ 過關要求

成功到達110ML水平層，坐上電動火車

🚂 第四章：110ML 運輸坑道

🏛️ 歷史場景 🎮 橫向衝刺 難度：⭐⭐⭐

🏗️ 場景描述

你終於到達 110ML 運輸坑道。呢度係海拔110米嘅主運輸層，坑道長約 2,200米，係礦場嘅心臟地帶。一架天線式電動牽引機車喺路軌上等你——每次可拖帶 10幾架載礦卡車，運載量達數十噸！

你坐上機車，拉動控制桿，列車沿路軌向洞口方向駛去。沿途經過支撐柱、通風管道、仲有豎井嘅底部出口。

洞口係半圓拱形，上面寫住「110M.L.」。出口設有 自動磅重儀 記錄產量。

🎮 玩法

  • 電動火車駕駛： 控制火車速度，避開路軌上嘅碎石
  • 換軌： 遇到分岔路要揀啱路線（有死路）
  • 通風扇： 經過大型通風扇時要低頭避開（快速㩒屏幕蹲下）
  • 收集礦石： 卡車上嘅鐵礦石會跌出嚟，要撿返

✅ 過關要求

成功駛出110ML洞口，磅重儀記錄到至少20噸礦石

🏭 第五章：選礦廠 → 碼頭區

🏛️ 歷史場景 🎮 搬運 + 限時 難度：⭐⭐

🏗️ 場景描述

礦石從110ML運出後，送到旁邊嘅 選礦廠（建於1954年，耗資十四萬元）。呢個廠房24小時運作，機器聲震耳欲聾！

選礦流程一路展開：

  1. 碎礦機 — 三組機械逐步將礦石打碎至直徑5厘米
  2. 振動篩 — 分離粗粒
  3. 乾選機（電磁） — 挑選高鐵礦粒
  4. 球磨機 + 棒磨機 — 磨成幼砂（不斷加水）
  5. 濕選機（電磁） — 吸起鐵砂，純度由37%提升至57-59%
  6. 脫磁 → 沉澱 → 真空脫水 — 變成精礦粉
  7. 運輸帶 — 送至精礦堆積場 → 再送去碼頭

🎮 玩法

  • 限時運輸： 喺指定時間內將礦石搬運到唔同嘅機器中
  • 小遊戲： 操作磁鐵吸起鐵砂，避開石頭（似夾公仔機）
  • 碼頭裝船： 用運輸帶將精礦送上躉船，再吊上貨輪「大寶山號」或「大寶石號」
  • 避車： 當年兒童返學要避開運礦車，遊戲中要避開來回嘅夾斗貨車

✅ 過關要求

成功將10噸精礦裝上貨輪，完成今日運送任務

🌍 礦工背景

馬鞍山嘅礦工嚟自 五湖四海——有湖南、湖北、山西、河北、遼寧嘅國民黨軍人/平民，亦有潮洲人、客家人、廣東人、台山人、順德人⋯⋯

佢哋啱啱到礦場嗰陣係 單身寡佬，住喺宿舍（俗稱 「工柵」）。後嚟結婚或者妻子兒女跟過嚟，就離開工柵，喺山頂區、馬鞍橋區、潮洲村區或碼頭區自己起木屋或簡單石屋，同家人一齊住。

🏘️ 第六章：礦村四區探索 + 天光墟

🏛️ 歷史場景 🎮 收集 + 解謎 難度：⭐

🏗️ 場景描述

完成礦場工作後，你決定探索呢個住了接近一萬人嘅礦村。礦村分為四個區域，每區住著唔同嘅社群：

📍 山頂區（北方人聚居）

呢度係礦村嘅經濟核心。有 信義會恩光堂（1952年建成，香港信義會第一所自建禮拜堂，正面山牆上有小鐘樓，歌德式尖拱窗，聖所後牆有「耶穌帶領羊群」壁畫）；天主堂聖若瑟堂（1952年，現代風格+歌德式尖拱門窗，混凝土結構，金字屋頂）；仲有 聖若瑟小學校、修道院、神父樓（入口有聖母抱聖嬰浮雕）。

📍 馬鞍橋區（客家人聚居）

位於馬鞍山郊野公園燒烤場附近。呢度有 馬鞍橋 遺跡，周圍係矮小石屋同菜園。

📍 上下半山/潮洲村區（潮州人聚居）

沿山路往下走，左邊路口進入就係潮州村。潮州工隊（如張松秋工隊）就係呢一帶。

📍 碼頭區（廣東人聚居）

即係今日嘅 恒安邨、耀安邨 位置。當年呢度有民居、車房、貨倉、木廠、仲有市集。礦石就係喺呢度用躉船同貨輪運去日本。

🎮 玩法

  • 探索收集： 喺四區走動，收集歷史文物（舊照片、礦工工具、教會文件）
  • 互動解謎： 教會場景有謎題要解（例如排列聖經經文、敲鐘順序）
  • 人物對話： 遇到村民可以對話，了解礦村生活故事
  • 小遊戲： 幫村民送貨、修理礦工屋企
  • 📸 拍照收集： 用手機鏡頭（遊戲內）影低歷史建築，解鎖圖鑑

可收集文物（參考真實歷史）： 信義學校舊課本、礦工頭燈、硝酸甘油炸藥包裝、大公洋行文件、日本工程師測量圖、礦車木剎車棍。

✅ 過關要求

收集至少8件文物，訪問3個村民，解鎖教堂隱藏房間

完成礦村探索後，你要由山頂安全到達山腳嘅 耀安邨。但呢個時候—— 每年雨季嘅洪水突然爆發！ 💀

🎮 BOSS戰玩法

  • 場景： 由鞍山探索館（山頂）一路向下跑到耀安邨停車場
  • 洪水追趕： 畫面底部洪水不斷上升，你要一路向上跳
  • 障礙物：
    • 🌊 海邊吹上嚟嘅鹹水風 — 強風會吹斜你，要𨅝低
    • 🚗 大佬亂cut線 — 的士突然從側邊衝出
    • 🚲 耀安邨單車仔 — 學生踩單車橫衝直撞
    • 🅿️ 停車場洪水 — 積水區要跳過，唔係會慢落嚟
    • 🪨 山泥傾瀉 — 礦場遺址嘅碎石滾落
    • 🚧 路陷 — 當年礦洞可能導致地面下陷

🎮 特殊機制

  • 體力值： 跑得太快會消耗體力，要收集路邊嘅「腐乳饅頭」補充（當年礦工嘅食物）
  • 分數加成： 路上有你之前收集嘅文物，撿到可以加分
  • 水位警示： 畫面邊緣會變色——由黃→橙→紅，代表洪水迫近

🏆 通關條件

成功逃到耀安邨安全位置，洪水淹浸前到達終點

🚀 天馬行空玩法設計

以下係一啲跳出傳統框架嘅 gameplay ideas，用嚟吸引玩家：

🌙 魚佬傳說

背景：魚佬為咗趕天光墟，前一晚 下晝就行路去西貢，喺碼頭等成晚，等到漁船返來買新鮮魚，然後摸黑行山路返馬鞍山。為咗 慳電蕊，佢唔捨得買新電筒電池，一路摸黑行路 🕯️

遊戲玩法係一個 資源管理 + 平衡 + 限時 survival：

🚶 **下晝出發**：由山頂行去西貢，要規劃路線
⏳ **等船時間**：喺西貢碼頭等漁船返來，期間可以同其他船民互動
🐟 **買魚**：有限金錢，要揀靚魚（睇魚眼、聞魚腥），講價！
🔦 **電筒電池**：得有限電量，要慳住用——開幾耐電筒、幾時摸黑行
🌧️ **隨機事件**：落雨路滑、遇到行山客、野狗追、跣親跌魚
🏆 **目標**：成功擔魚返到天光墟開檔，魚要新鮮、人要準時！

靈感源自 Ming 提供嘅真實故事：魚佬為慳電蕊，摸黑行山路 🥹

🔔 教堂鐘聲節奏遊戲

扮演神父/牧師/修女，敲教堂嘅鐘！

🎵 跟住節奏㩒屏幕，敲出唔同嘅鐘聲
📅 平日敲3下、星期日敲7下、聖誕節敲12下
💥 敲錯會俾教友投訴
🏆 解鎖隱藏鐘聲旋律（可以敲出《奇異恩典》！）

🍬 士多買糖誘惑戰

扮演礦工小朋友，每日經過士多嘅大窗口——忍住唔買糖！

🍭 玻璃樽入面嘅彩色糖果會閃閃發光引誘你
💰 每日零用錢有限，買咗糖就儲唔到錢買心儀嘅玩具
🦷 食太多糖會蛀牙，要去診所睇醫生（修女幫你補牙）
🧠 考驗玩家嘅自制力——真正嘅「兒童理財教育」！

🦜 金絲雀伴侶系統

遊戲開始時可以領養一隻金絲雀做寵物！

🐦 金絲雀會幫你探測毒氣（礦洞關卡需要）
🎵 可以學佢唱歌，解鎖隱藏通道
💕 要定時餵食、清潔，唔係會唔開心
👀 金絲雀有時會發現隱藏物品（閃閃發光嘅鐵礦石晶體）
🌟 全遊戲最有靈魂嘅 companion！

⏳ 一日生存模式

遊戲以「一日」為單位，有晨、晝、昏、夜四個時段：

🌅 **清晨6:00** — 天光墟買餸/開檔
☀️ **日間** — 礦場工作/教會教學/返學
🌆 **黃昏** — 放工、礦村活動、做功課
🌙 **夜晚** — 家庭時間、教會活動、秘密探索

每個時段有固定時間，玩家要分配時間做唔同任務。夜晚可以探索平時去唔到嘅地方……
