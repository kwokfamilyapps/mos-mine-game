## 🎬 第1段：開場動畫（15秒）
**位置：** 遊戲開場
**風格：** 暖金晨光、懷舊紀錄片

### 中文 Prompt：
```
1960年代香港馬鞍山，清晨霧氣籠罩山巒。
無人機高空航拍視角，俯瞰整片馬鞍山山脈。
金色晨光照亮山腰上一個礦村，到處是矮小的石屋。
鏡頭緩慢下降，越過山頂區的一座教堂（信義會恩光堂），屋頂有十字架和小鐘樓。
飛過一條石橋（平安橋），橋頭豎立一塊石碑，上面直寫「平安槁」三個字。
最後鏡頭推進到一個混凝土礦洞口，拱門上方寫著「240M.L.」。
標題文字浮現：「馬鞍山鐵礦場 — 1960年代」。
溫暖懷舊的菲林質感，紀錄片風格。
背景音樂：口琴與風聲。
```

## 🚶 第2段：平安橋 → 警衛室（10秒）
**位置：** 第一章→第二章過場
**風格：** 清晨薄霧、生活氣息

### 中文 Prompt：
```
1960年代香港馬鞍山，清晨薄霧籠罩礦村。
一個石頭石碑佇立在橋頭，石碑上刻有豎排的「平安槁」三個字。
鏡頭切換到一個礦工，身穿藍色工衣，走進礦場警衛室。
警衛室牆上有一塊通告板，掛滿了紅綠兩色的名牌。
礦工找到自己的名牌，從綠色翻轉成紅色（代表已進入礦洞）。
他拿起礦工頭燈，轉身走向黑暗的礦洞口。
柔和的晨光，腳步聲在碎石路上迴盪。
電影感的變形寬銀幕鏡頭，溫暖色調。
```

## ⛏️ 第3段：礦洞穿越四層（12秒）
**位置：** 240ML → 192ML → 144ML → 110ML
**風格：** 礦洞燈光、工業機械

### 中文 Prompt：
```
1960年代香港馬鞍山鐵礦場地下坑道內部。
一架裝滿鐵礦石的礦車開始沿著斜路向下滑行。
鏡頭跟著礦車，經過多個水平層——牆上標示著「240ML」。
到達192ML平台，礦工推著載滿礦石的手推車經過。
繼續下降到144ML平台，礦工用風鑽鑽孔、用十字鎬挖掘礦石。
礦車抵達垂直豎井，然後向下墜落，火花四濺。
燈光閃爍，最終降落在110ML的電動火車旁邊。
灰塵在昏暗燈光中飄浮，工業回聲。
低角度動態追蹤鏡頭。
```

## 🚂 第4段：火車衝出礦洞（10秒）
**位置：** 110ML坑道 → 選礦廠
**風格：** 速度感、工業機械

### 中文 Prompt：
```
1960年代馬鞍山鐵礦場110ML運輸坑道內部。
一架架空電纜式電動機車啟動，後面拖著10卡載滿礦石的卡車。
鏡頭安裝在火車頭前面，向前衝刺穿過長達2,200米的坑道。
架空電纜擦出火花，通風扇在頭頂轉動。
火車駛向洞口——一個半圓拱形的混凝土出口，上方刻著「110M.L.」。
強烈的日光湧入，火車衝出洞口到達地面。
前方是一座選礦廠（建於1954年），運輸帶和粉碎機在運轉。
蒸汽鳴笛，金屬摩擦聲，火車汽笛迴盪。
寬闊的電影感建立鏡頭。
```

## 🏭 第5段：選礦廠 → 碼頭裝船（10秒）
**位置：** 選礦廠 → 碼頭
**風格：** 工業勞動、24小時運作

### 中文 Prompt：
```
1950年代香港馬鞍山選礦廠內部，24小時不斷運作。
巨大的碎石機將礦石打碎，運輸帶不斷運送鐵礦砂。
電磁選礦機分離鐵礦顆粒，水流沖刷過球磨機和棒磨機。
鏡頭跟著運輸帶，將加工好的鐵礦砂送往戶外。
運輸帶一路延伸到碼頭，一艘貨輪停泊在岸邊。
鐵礦砂傾倒進貨艙，船身標示著「大寶山號」。
工人忙著裝載物資，蒸汽從機器上升起。
工業紀錄片風格，動態鏡頭運動，溫暖的機油色調。
```

一個魚佬擔著兩擔裝滿鮮魚的竹籮，從西貢行山路返來，一臉疲憊但帶著笑容。
村民忙著擺檔：菜檔、豬肉檔、一個外省男人在即烤燒餅，香氣四溢。
穿校服的礦工小孩經過士多大窗，裡面玻璃樽裝滿彩色糖果，小朋友忍不住多看兩眼。
山頂教堂傳來鐘聲——鏡頭推向教堂鐘樓特寫。
村民們越過平安橋去開工，晨光穿透雲層。
溫馨懷舊的氣氛，鳥鳴聲。
