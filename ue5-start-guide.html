📌 開工前準備：
  • ✅ Unreal Engine 5.4+ 已安裝
  • ✅ 開新 Project：選 Blank → Third Person → Mobile/Tablet
  • ✅ 建議開新 Folder：C:\UE5_MineTrain

Phase 1：Spline 路軌 + 礦車（今日開工）

第1步：開新 Project
打開 UE5 → New Project → Games → Blank → 選「With Starter Content」→ Blueprint → Project Location: 你自己揀位 → Create
第2步：建立 Spline 路軌 Blueprint
1. 喺 Content Browser ➡ 右鍵 ➡ Blueprint Class ➡ 揀 Actor
2. 改名：BP_TrackSpline
3. 雙擊打開 BP_TrackSpline
4. 左邊 Components 面板 ➡ Add Component ➡ 揀 Spline Component
5. 改名：TrackSpline
// 跟住去 Event Graph
// 未需要寫 code，住要個 spline 做主幹
第3步：建立礦車 Blueprint
1. Content Browser ➡ 右鍵 ➡ Blueprint Class ➡ 揀 Pawn
2. 改名：BP_MineCart
3. 雙擊打開，Components 加以下：

Component設定
StaticMesh揀 Shape_QuadPyramid 做車身（暫用）
SpringArm位置 Z +150
Camera跟 SpringArm
SphereCollisionRadius 50，做碰撞
第4步：寫礦車移動邏輯（核心！）
打開 BP_MineCart 嘅 Event Graph，跟住整：
// === VARIABLES（變量）=== // 類型: Actor 名稱: TrackSplineRef 預設: None // 類型: Float 名稱: Speed 預設: 0 // 類型: Float 名稱: MaxSpeed 預設: 500 // 類型: Float 名稱: DistanceAlong 預設: 0 // 類型: Boolean 名稱: bIsMoving 預設: false // === EVENT TICK === 1. 喺 Event Graph 空白位右鍵 ➡ 打 "Tick" ➡ 揀 Event Tick 2. 拉出 Event Tick ➡ 右鍵 ➡ 打 "Branch" 3. 拉出 Branch Condition ➡ 右鍵 ➡ 打 "bIsMoving" 4. 拉出 Branch True ➡ 右鍵 ➡ 打 "Get Distance Along Spline" 5. 拉出 TrackSplineRef 個 pin ➡ 右鍵 ➡ 打 "Spline Length" 6. 拉出 Get Distance Along Spline Return Value ➡ + ➡ Speed * Delta Time ➡ ➡ 做 Clamp (min:0, max:[Spline Length]) 7. 結果 Set Distance Along Spline 8. 拉出 TrackSplineRef ➡ 右鍵 ➡ 打 "Get Location at Distance Along Spline" ➡ 連接 Distance 同 Return Value 9. 拉出 Get Location ➡ 右鍵 ➡ 打 "Set Actor Location"
第5步：鍵盤控制加減速
同樣喺 BP_MineCart Event Graph：
// 加速（W鍵） 1. Event Graph 空白 ➡ 右鍵 ➡ "Keyboard Event W"（或 "InputAction 加速"） 2. 拉出 Pressed pin ➡ Sequence ➡ Then 0: ➡ 打 "bIsMoving = true" ➡ Then 1: ➡ 做 "Speed = MaxSpeed" // 減速（S鍵） 3. Keyboard Event S ➡ Pressed ➡ "Speed = MaxSpeed * 0.3"（慢速） ➡ Released ➡ "Speed = 0" + "bIsMoving = false" // 調校速度 建議初始 MaxSpeed = 300（慢速觀光） 將來可以整油門/煞車 analog 控制
第6步：拉路軌落 Level
1. 㩒返 Content Browser → 將 BP_TrackSpline 拉去 Level
2. 選中 BP_TrackSpline → Details → Spline 嘅 Spline Points
3. 㩒 + 加點，逐個拉出蜿蜒路軌
4. 最少 5-6 個點，扭少少彎
5. 將 BP_MineCart 都拉落 Level
6. 選中 BP_MineCart → Details → TrackSplineRef → 㩒 ▶ 揀返你條路軌
7. Play 測試！W 加速，S 減速
第7步：整條靚仔啲嘅路軌
用 Static Mesh 整 rails 同 sleepers：
// Rail Mesh 喺 BP_TrackSpline ➡ Add Component ➡ SplineMesh Component 改名：RailMeshes // 搵個長條形 mesh 做路軌 // 可用 Starter Content 嘅 Shape_Cube (scale 拉長) // 或者市場下載 railway track mesh // 用 Spline Mesh 沿 Spline 鋪路軌： Event BeginPlay ➡ ➡ 用 For Loop 行 spline points ➡ 每兩個 point 之間放一條 SplineMesh ➡ Set Start/End Position & Tangent

Phase 2：地形 + 場景（第2日）

第1步：起 Landscape 地形
1. Toolbar ➡ Landscape Mode
2. 揀 New Landscape
3. Section Size: 7x7 (大約 255x255)
4. 用 Sculpt Tool 整山丘、斜坡
5. 建議：路軌起點（240ML）要高，終點（耀安邨）要低
6. Paint 草地材質
第2步：場景建築擺位
用 Starter Content 嘅 Box / Cylinder 暫代建築物：
建築大概位置暫用 Mesh
240ML 礦洞入口起點（地勢高處）Box 拼成方形入口
選礦廠中途Box 組合
恩光堂礦村區Box 做主體 + Cone做尖頂
天光墟恩光堂門外小路 + Box 做檔口
平安橋山澗位置Box 做橋面
礦村屋沿路散落Box 組合
耀安邨碼頭終點（地勢低處）平面+Box 做碼頭
💡 貼士：先用 Box 拼出大概位置，確認路線同距離啱先，唔使一開始就整靚模型。
